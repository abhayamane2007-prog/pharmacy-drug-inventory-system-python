#  Pharmacy Drug Inventory System (Python)

A simple command-line Pharmacy Drug Inventory System developed using Python and Linked Lists.

## Features

-  Add Medicine
-  Display Medicines
-  Search Medicine
-  Delete Medicine
-  Check Expired Medicines
-  Near Expiry Alerts



## Sample Menu

```
========== Pharmacy Drug Inventory System ==========

1. Add Medicine
2. Display Medicines
3. Search Medicine
4. Delete Medicine
5. Check Expiry Alerts
6. Exit
```
